SWE 363: Web Engineering & Development (3-0-3)

Syllabus – Summer Semester 2019-2020 (193)

“Summer Innovative Teaching Program“

The nature of web application development in real-world work is to be conducted mostly by teams. Moreover, students learn a lot from their peers. During the summer semester, the course will adopt the innovative approach of inquiry-based learning where real cases are presented, plans are discussed, solutions and resources are explored, and ideas and reflections are shared. The course will emphasize this practice further by encouraging collaborative groups. Besides online in-class discussions, students are expected to work in monitored teams to gain technical knowledge and skills to create reliable secure database-driven and appealing hybrid web applications that are search-engine optimized and crossplatform compatible that automatically adjust to look good on all devices from smart phones to desktop computers. No major or final exams, but to assure the attainment of the course outcomes, assessment will be composed of individual short quizzes and portfolios in addition to walk-through team projects and presentations.

Course Catalog Description
Internet basics for web applications. Web Engineering fundamentals: requirements, analysis modeling, design modeling, testing. Technologies and tools for developing web applications: markup languages, styling, client and server side programming, data description and transformation. Web services. Advances in web engineering

Pre-requisites: Junior Standing

Course Objectives

    Provide students with conceptual and practical knowledge and skills required to develop web applications and web services.

Provide students with conceptual and practical knowledge and skills required to develop web applications and web services.

    Upon completion of the course, you should be able to:
  1. Perform analysis modeling and design modeling for web applications
  2. Identify candidate tools and technologies for developing web applications.
  3. Develop user-interfaces for web applications
  4. Describe and transform data using XML and its related technologies.
  5. Develop web applications and web services.

Content Breakdown:

  • Introduction to Web Application Development and Engineering (1 week)
    - Basics of WWW: Components and layered architecture
    - Categories of Web applications and user-driven web experience
    - Overview of Web application technologies and frameworks
    - Web development cycle and design criteria

  • Front-end client-side programming (3 weeks)
    - Responsive Web Design
    - Content and Layout Structure and Styling
    - Search Engine Optimization
    - Interactive Websites & Error Handling

  • Back-end server-side programming (3 weeks)
    - Dynamic Pages and Web Services
    - Error Handling and Validation
    - Database-driven Interactive Applications
    - Handling Security Issues
